# Changelog

All notable changes to this project are tracked in git tags. The package version is derived from tags via `setuptools_scm`, and tags are bumped automatically on every push to `main` by `.github/workflows/auto-tag.yml`, which reads the `#none` / `#patch` / `#minor` / `#major` marker from the merge commit subject.

For the full history, see the [GitHub releases page](https://github.com/TimoIllusion/inference-streaming-benchmark/releases) and `git log`.

This file follows the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format. Significant releases may be noted here in addition to the auto-generated tags.
